Features

Ever wanted to chat on Discord while on an out-of-support machine? Well, you're in luck!

With Discord Messenger, you can enjoy the same experience as with the official Discord client*.
  • Doesn't run in a glorified web browser
  • Runs on Windows 2000, XP, Server 2003, and everything that came after
  • Entirely compatible with Discord, so you can chat with your friends as usual
  • Open-source under a free license forever
  • Source is easily expandable
  • Runs in the background, so you never miss a message
  • Easy to customize with a background image or four predefined styles
  • Share images and other files easily
  • Lightweight and optimized to run smoothly on old, creaky hardware
  • Streamlined user interface so you can get where you need in just a few clicks
  • Powerful quick switcher to surf around Discord with ease
  • Supports high DPI monitors
  • No tracking or analytics APIs used
  • In-built update checker, which can be disabled if you don't care
  • No install needed, just download and open!
* - Text channels only. Voice & video is currently not supported.
